Telegram bot, that predicts your future once a year

### Development
#### Run local in docker
```shell
> docker compose up
```
#### Install dependencies
```shell
> # activate venv
> make install
```
#### Run test
```shell
> make test
```
#### Update requirements
```shell
> # change pyproject.toml
> make requirements.txt
```
#### Sync requirements with requirements.txt
```shell
> make sync
```
